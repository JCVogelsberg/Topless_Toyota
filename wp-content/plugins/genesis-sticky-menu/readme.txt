== Description ==

This plugin adds a sticky menu. When you scroll down, the sticky menu will appear at the top. Now scroll back to the top of the page, and youâ€™ll see the sticky menu disappear. This plugin will create Third menu location named "Genesis Sticky menu". Select your menu on this location. You can choose Background color, Background hover color, Text color and Text hover color

Requires the Genesis framework. 

Check <a href="http://iniyan.in/">plugin demo here</a>

*I really thank <a href="http://logeshkumar.com">Logeshkumar </a> for his cooperation.* 

== Installation ==

1. Upload the entire `genesis-sticky-menu` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To Select the sticky menu Appearance->Menu-> Select your menu on theme location named "Genesis Sticky menu" 
4. Make your design changes. Genesis -> Sticky menu

== Frequently Asked Questions ==

= The plugin won't activate =

You must have Genesis or a Genesis child theme installed and activated on your site.
